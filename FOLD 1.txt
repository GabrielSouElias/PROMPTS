# FOLD 1 — HERO | Adega Própria Landing Page

---

## Role & Design Philosophy

You are a senior UI designer and frontend developer building the **hero section** of a luxury wine brand landing page. This is the single most important fold — it sets the emotional tone for the entire page and must convert first-time visitors within seconds.

Build a premium, **wine-dark** immersive hero that sells a **transformation**: the dream of owning your own premium wine brand with wines sourced directly from renowned Serra Gaúcha wineries. This is NOT a product page. This is an emotional entry point — the visitor should feel like they just pushed open the heavy wooden door of an exclusive wine cellar at night, and stepped into darkness lit only by candlelight and the amber glow of aged oak.

**The darkness is the design — and the darkness is wine-red, not green.** The hero must be overwhelmingly dark. `#1A0F10` (deep wine-black) is the enveloping canvas. `#2A1416` is the shadow layer. `#8C3D20` and `#A62C21` are the warm undercurrents of depth. Gold (`#BF8C60`) and beige (`#D9BBA9`) appear only as rare, precious details — like candlelight reflected on the curve of a wine glass. If you sampled the hero's pixel colors into a histogram, 85%+ should be in the wine-dark range. The light exists to serve the darkness, not to compete with it.

**Absolutely avoid any green-led interpretation.** The previous version of this design drifted toward green. This must NOT happen. The canvas is `#1A0F10` — wine-black, not green-black.

**Design mandates (non-negotiable):**
- No emoji icons. No inline styles (except for complex radial gradients Tailwind cannot express). No generic gradients.
- No placeholder images except dark, moody Unsplash wine photography with heavy dark overlay treatments.
- Typography must be intentional — use the established Fraunces + DM Sans pairing.
- Every element must feel like it belongs in an Awwwards-winning portfolio.
- Mobile-first: design for 375px, then enhance for desktop.
- The hero must pass the "screenshot test" — if posted on Dribbble, it receives genuine engagement.

## Layout Architecture

The hero is a **full-viewport immersive section** (`min-h-[100dvh]`). It is composed of **six visual layers** stacked via `position: relative` / `absolute`, creating a deep sense of dimensional wine-dark atmosphere:

### Layer 1 — Parallax Background Image (deepest)

A full-bleed background image (dark wine cellar / vineyard at night) that moves vertically on scroll via Framer Motion's `useScroll` + `useTransform`. The image is `position: fixed` (desktop) or `position: absolute` (mobile — disable parallax for performance). Wrapped in a `clipPath: polygon(0% 0, 100% 0%, 100% 100%, 0 100%)` container to prevent the fixed image from bleeding into adjacent folds.

**Reference the Animation Background component in `/COMPONENTES/ANIMATION BACKGROUND/`.** Study the `demo.tsx` file — it demonstrates the exact `useScroll` + `useTransform` + `clipPath` + `fixed` positioning technique. Adapt it: replace the Next.js `<Image>` with a standard `<img>` tag + absolute positioning, and apply the brand's heavy dark overlay treatment on top.

**CRITICAL IMAGE TREATMENT:** The background image must be dramatically darkened. It should be barely visible — a texture of shadows and shapes more than a photograph. The image provides depth and organic texture to the darkness, NOT visual content. Apply a CSS `brightness(0.3)` filter on the `<img>` itself (`brightness(0.25)` on mobile) PLUS the overlay stack below. The visitor should feel the image more than see it.

### Layer 2 — Wine-Dark Overlay Stack (where the mood lives)

Multiple overlapping gradient and color layers that transform the image into a branded wine-dark atmosphere. These are the most important elements in the entire hero — they create the wine cellar mood and enforce the wine-red (not green) identity:

**2a. Primary anchor gradient (heaviest):**
`absolute inset-0` with inline style `background: linear-gradient(to top, #1A0F10 0%, rgba(26,15,16,0.92) 40%, rgba(26,15,16,0.55) 70%, transparent 100%)`
This grounds the bottom 60% of the hero in near-solid wine-black. All text at the bottom half has perfect readability.

**2b. Top shadow layer (uses #2A1416 — the dark red-black):**
`absolute top-0 left-0 right-0 h-[40vh]` with inline style `background: linear-gradient(to bottom, rgba(42,20,22,0.75) 0%, transparent 100%)`
Darkens the top zone for navigation readability using the #2A1416 shadow layer, reinforcing the wine-dark identity.

**2c. Radial cinematic vignette (pushes edges to wine-black):**
`absolute inset-0` with inline style `background: radial-gradient(ellipse at center, transparent 30%, #1A0F10 85%)`
A powerful radial vignette that pushes the edges into near-blackness, focusing the eye toward the center where the headline lives. The `30% → 85%` range is tighter than standard — this is cinematic, not photographic.

**2d. Wine-red atmospheric wash (lower center):**
`absolute inset-0` with inline style `background: radial-gradient(ellipse at 50% 70%, rgba(166,44,33,0.08) 0%, transparent 50%)`
A barely-perceptible wash of deep wine red in the lower-center. Adds a warm, organic undertone like the ambient reflection of red wine in a dim room. At 0.08 opacity (Master's upper limit), it's felt more than seen.

**2e. Clay warmth (mid-layer, off-center-left — the oak barrel reflection):**
`absolute inset-0` with inline style `background: radial-gradient(ellipse at 30% 40%, rgba(140,61,32,0.05) 0%, transparent 40%)`
An extremely subtle reddish-brown warmth positioned slightly off-center-left. This prevents the darkness from feeling cold or clinical — it adds the warmth of aged oak barrels and keeps the overall tone firmly in wine territory.

**2f. Gold headline glow (the "candle" in the cellar):**
`absolute top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2 w-[500px] h-[350px] lg:w-[700px] lg:h-[450px] pointer-events-none` with inline style `background: radial-gradient(ellipse at center, rgba(191,140,96,0.06) 0%, transparent 65%)`
This is the one light source — a soft, barely-there golden glow positioned behind the headline. It creates the subliminal sensation of illumination on the words without being visible as a shape. At 0.06 opacity, it should NOT be identifiable as "there's a glow there" — it should feel like the headline area is naturally slightly warmer.

### Layer 3 — Grain Texture Overlay

Procedural SVG noise that adds analog richness. The difference between a digital flat surface and textured paper:

```jsx
<svg className="absolute inset-0 w-full h-full pointer-events-none opacity-[0.035] mix-blend-overlay" xmlns="http://www.w3.org/2000/svg">
  <filter id="heroGrain">
    <feTurbulence type="fractalNoise" baseFrequency="0.65" numOctaves="3" stitchTiles="stitch"/>
  </filter>
  <rect width="100%" height="100%" filter="url(#heroGrain)"/>
</svg>
```

Opacity at `0.035` to keep the grain ultra-subtle against the wine-dark background. The grain should add texture to dark surfaces without lightening them.

### Layer 4 — Navigation (fixed top)

The navigation bar sits at the top with its own background treatment that transitions on scroll. See the Navigation section below for full specification.

### Layer 5 — Content (topmost — z-10)

All text and CTA elements. The content is the ONLY place where light colors appear prominently. Text should feel like it's glowing softly out of the darkness — luminous words floating in shadow.

## Animation & Motion Specification

### Parallax Background (desktop only, reduced-motion aware)

```jsx
const [isMobile, setIsMobile] = useState(true);
const [prefersReducedMotion, setPrefersReducedMotion] = useState(false);

useEffect(() => {
  setIsMobile(window.innerWidth < 1024);
  setPrefersReducedMotion(window.matchMedia('(prefers-reduced-motion: reduce)').matches);
  
  const handleResize = () => setIsMobile(window.innerWidth < 1024);
  window.addEventListener('resize', handleResize);
  return () => window.removeEventListener('resize', handleResize);
}, []);

const heroRef = useRef(null);
const { scrollYProgress } = useScroll({
  target: heroRef,
  offset: ["start start", "end start"]
});
const backgroundY = useTransform(scrollYProgress, [0, 1], ["0%", "25%"]);
const enableParallax = !isMobile && !prefersReducedMotion;
```

Apply `backgroundY` as the `y` style on `<motion.div>` wrapping the image — ONLY when `enableParallax` is true. On mobile or reduced-motion, render a static `absolute` image with no transforms.

### Nav Scroll Detection

```jsx
const [navScrolled, setNavScrolled] = useState(false);

useEffect(() => {
  let ticking = false;
  const handleScroll = () => {
    if (!ticking) {
      requestAnimationFrame(() => {
        setNavScrolled(window.scrollY > 80);
        ticking = false;
      });
      ticking = true;
    }
  };
  window.addEventListener('scroll', handleScroll, { passive: true });
  return () => window.removeEventListener('scroll', handleScroll);
}, []);
```

Apply to nav via conditional class.

### Content Entrance Choreography

All content elements animate on mount — the hero is above the fold. Use Framer Motion with staggered delays:

| Element | Delay | Duration | Initial State | Animate To | Easing |
|---|---|---|---|---|---|
| Nav (logo + CTA) | 0ms | 800ms | `opacity: 0, y: -15` | `opacity: 1, y: 0` | `[0.22, 1, 0.36, 1]` |
| Eyebrow label | 200ms | 700ms | `opacity: 0, y: 25` | `opacity: 1, y: 0` | `[0.22, 1, 0.36, 1]` |
| Headline | 400ms | 900ms | `opacity: 0, y: 35` | `opacity: 1, y: 0` | `[0.22, 1, 0.36, 1]` |
| Subheadline | 600ms | 700ms | `opacity: 0, y: 25` | `opacity: 1, y: 0` | `[0.22, 1, 0.36, 1]` |
| CTA button | 800ms | 700ms | `opacity: 0, y: 15, scale: 0.96` | `opacity: 1, y: 0, scale: 1` | `[0.22, 1, 0.36, 1]` |
| Micro-text | 950ms | 600ms | `opacity: 0` | `opacity: 1` | `easeOut` |
| Scroll indicator | 1300ms | 600ms | `opacity: 0, y: 8` | `opacity: 1, y: 0` | `easeOut` |
| Gold hairline | 1500ms | 900ms | `scaleX: 0` | `scaleX: 1` | `[0.22, 1, 0.36, 1]` |

The easing `[0.22, 1, 0.36, 1]` is the project's signature cubic-bezier — used across ALL folds for visual consistency.

Gold hairline uses `style={{ transformOrigin: 'center' }}` — line draws from center outward.

**On mobile:** Simplify all entrances to opacity-only fades (remove `y` translations) to avoid perceived layout shifts on slower devices. Keep stagger timing.

**If `prefersReducedMotion` is true:** Skip all entrance animations — render at final state immediately. No parallax, no stagger, no bouncing indicator.

### Framer Motion Implementation Pattern

```jsx
<motion.h1
  initial={prefersReducedMotion ? false : { opacity: 0, y: isMobile ? 0 : 35 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.9, delay: 0.4, ease: [0.22, 1, 0.36, 1] }}
>
```

Setting `initial` to `false` under reduced motion skips the animation entirely.
