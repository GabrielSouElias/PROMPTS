## COLOR SYSTEM

Use this as the core palette:

- **#1A0F10** — deep wine-black base
- **#2A1416** — dark red-black shadow layer
- **#8C3D20** — reddish brown
- **#A62C21** — deep wine red
- **#BF8C60** — burnished gold-beige
- **#D9BBA9** — light warm beige

### Color Hierarchy
The page must be **dark-first**.

#### Dominant colors
- #1A0F10
- #2A1416
- #8C3D20
- #A62C21

These should carry the majority of the page’s visual weight.

#### Accent-only colors
- #BF8C60
- #D9BBA9

These should be used sparingly for:
- CTA emphasis
- text highlights
- elegant border lines
- subtle glow accents
- selected italic emphasis words
- premium micro-details

### Updated Visual Weight Distribution
Respect this distribution across the interface:

- #1A0F10 → **45–55%**
- #2A1416 → **20–25%**
- #8C3D20 → **12–18%**
- #A62C21 → **8–12%**
- #BF8C60 → **5–8% maximum**
- #D9BBA9 → **text and subtle detail use only**

### Color Rules
- Never use #BF8C60 or #D9BBA9 as large background fills
- Gold and beige must never dominate large surfaces
- Surfaces/cards must remain dark, only slightly elevated from the base
- Gold should feel precious, not abundant
- Beige should primarily serve text hierarchy and subtle UI support
- The dominant emotional impression should be dark wine, shadow, oak warmth, and restrained gold light
- Avoid any layout that reads primarily as green
- Avoid cool emerald dominance
- The darkness should dominate at every scroll position

### Gradient Rules
Section gradients must transition between dark tones only, such as:
- #1A0F10 → #2A1416
- #1A0F10 → #8C3D20
- #1A0F10 → #A62C21
- #2A1416 → #8C3D20
- #1A0F10 → #8C3D20 → #1A0F10
- #1A0F10 → #A62C21 → #1A0F10

Do not use gold-based gradients as major section backgrounds.

### Atmospheric Glow Rules
Atmospheric glows must be extremely subtle.
Use low opacity only:
- **0.04 to 0.10 maximum**

Good use cases:
- faint wine-red bloom
- muted brown-red warmth
- extremely soft gold light behind focal text or CTA
- low warm glow near section seams

Avoid:
- neon glows
- bright halo effects
- obvious color fog
- strong emerald ambience
- tech-like lighting

### Border Rules
Borders must be nearly invisible.

Preferred border logic:
- border-[#BF8C60]/10
- border-[#D9BBA9]/5
- border-[#8C3D20]/20

Borders should be:
**felt more than seen**

### Surface Strategy
- Primary canvas should use #1A0F10
- Secondary depth layers should use #2A1416, #8C3D20, and restrained #A62C21
- Elevated surfaces must remain dark
- No large pale or gold cards
- No green-led surfaces unless used only as a buried shadow undertone, never as the main identity

### Text Color Strategy
#### Headline text
Use #D9BBA9 as the main headline color in full opacity where needed.

#### Body text
Use #D9BBA9 at reduced opacity for supporting text:
- #D9BBA9/75
- #D9BBA9/70

#### Muted text
Use:
- #D9BBA9/50
- #D9BBA9/45

#### Accent text
Use #BF8C60 for:
- emphasis words
- italic serif highlights
- premium labels
- subtle metadata
- selected CTA accents

Do not use bright white as a dominant text color.
Do not use cool gray text.
Do not introduce off-palette font colors.

### Atmosphere Metaphor
Think:
**a candlelit wine cellar where darkness dominates and gold appears only in whispers**

## TYPOGRAPHY DIRECTION

Never use:
- Inter
- Roboto
- Arial
- generic default sans-serifs

Use a luxury pairing that feels refined and editorial.

### Recommended Direction
#### Display / Headline Font
Choose one elegant serif such as:
- Cormorant Garamond
- Playfair Display
- DM Serif Display
- Fraunces
- Instrument Serif

#### Body / UI Font
Choose one modern supporting font such as:
- DM Sans
- Outfit
- Manrope
- General Sans equivalent if available
- another clean modern sans that does not feel generic

### Typographic Style Rules
- large headlines should feel sculpted and intentional
- small labels should use spaced uppercase tracking
- one emphasis word per major headline may use italic serif styling
- that emphasis word may use the gold accent color
- body copy should be lighter in weight than headlines
- avoid dense paragraphs
- keep copy surgical and emotionally charged

### Typography Color Consistency
Typography must follow the darker, more sophisticated visual direction:
- headlines should feel warm and elegant, never stark white
- body text should feel soft and premium, never gray-tech or cool-toned
- accent words should use gold selectively and elegantly
- muted text should remain readable without flattening the atmosphere
- the entire text system must harmonize with dark wine tones, not contrast against them harshly

## MOBILE-FIRST PRIORITY

This is a **mobile-first landing page**.

Design for **375px width first**, then scale upward.

### Mandatory Mobile Rules
- no horizontal scroll ever
- minimum 44x44 tap targets
- body text minimum 16px on mobile
- CTA buttons must be highly tappable
- hover-only interactions must have touch-safe equivalents
- animations must remain performant on mobile
- parallax and sticky behaviors must gracefully simplify on smaller screens
- avoid heavy desktop-only motion logic on mobile hardware
- use transforms and opacity whenever possible
- respect reduced motion where appropriate

Do not treat mobile as a compressed desktop version.
